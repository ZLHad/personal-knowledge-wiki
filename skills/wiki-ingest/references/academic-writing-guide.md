# Academic Writing Extraction Guide

## Focus

The value of academic writing conversations lies in **reusability of revision experience**. Every polish/revision contains extractable universal rules.

## Raw Record Structure

## Wiki Update Guidance

From academic writing conversations, typically update:
- **Concept pages**: Newly discovered writing rules/principles → update or create `wiki/concepts/`
- **Topic pages**: Specific section/dimension norms → update `wiki/topics/`
- **Entity pages**: Journals/tools involved → update `wiki/entities/`

## Routing Guide

| Information Type | Where to File |
|-----------------|---------------|
| New case for an existing rule | Update corresponding topic page |
| Brand new writing principle | May deserve a new concept page |
| Journal formatting preference | Update corresponding entity page |
| Personal habit/preference | Keep in raw record only |
